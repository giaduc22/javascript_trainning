# Function

## Function declarations

- Can be hoisted

```javascript
    function name([param[, param[, ... param]]]) {
        statements
    }
```
## Function expressions

- Can be name or unnamed
- Not hoisted

```javascript
    let myFunction = function [name]([param1[, param2[, ..., paramN]]]) {
        statements
    };
```

## Arrow function

```javascript
    ([param[, param]]) => {
        statements
    }
```
